# Credit Card Fraud Detection System

This project is a Machine Learningâ€“based web application that predicts whether a credit card transaction is **Legitimate** or **Fraudulent**.  
The model is trained on anonymized transaction data and deployed using **Streamlit** for real-time prediction.

---

## Project Overview

Credit card fraud detection is a real-world classification problem in the financial domain.  
This project focuses on building an end-to-end ML pipeline, from model training to deployment, to identify fraudulent transactions efficiently.

---

## Tech Stack

- Python  
- NumPy  
- Pandas  
- Scikit-learn  
- Streamlit  
- Pickle (Model Serialization)

---

## Machine Learning Workflow

- Data preprocessing and feature scaling  
- Training a supervised classification model  
- Model evaluation using classification metrics  
- Saving the trained model using Pickle  
- Deploying the model as a web application using Streamlit  

---

## Web Application Features

- Clean and simple user interface  
- One-click prediction (no need to manually enter all features)  
- Displays transaction risk with confidence score  
- Real-time fraud detection simulation  

---

## Deployment

The application is deployed using **Streamlit Community Cloud** and can be accessed directly via a web browser.
